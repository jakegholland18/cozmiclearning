# CozmicLearning Promotional Video Script - Synthesia Production

**Target Length**: 60-90 seconds
**Style**: Energetic, motivational, Pixar-inspired
**Character Consistency**: Use SAME avatar ID for all scenes

---

## PRODUCTION NOTES FOR SYNTHESIA

### Character Consistency Strategy:
1. **Choose ONE avatar** at the start (recommend: young male avatar, energetic style)
2. **Use that same avatar ID for ALL scenes**
3. **Don't regenerate or change avatars between scenes**
4. **Use background changes and camera angles** to show scene transitions
5. **Add text overlays and graphics** to enhance storytelling without changing the character

### Recommended Synthesia Settings:
- **Avatar**: Same ID throughout (e.g., "James" or "Liam" - young male avatar)
- **Voice**: Upbeat, energetic male voice (recommend: Joey or Matthew)
- **Pace**: Fast (1.2x speed for energy)
- **Background**: Change per scene (will provide)
- **Camera**: Medium close-up throughout for consistency
